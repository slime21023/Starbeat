Starbeat uses Markdown for content. You can write your documentation in `.md` files.

## Features

- **Markdown Support**: Write in standard Markdown
- **Code Highlighting**: Syntax highlighting for code blocks
- **Frontmatter**: Configure page metadata with YAML

## Example

```markdown
---
title: My Page
order: 1
group: Getting Started
---

# Hello World

This is my documentation page.
```
