Welcome to Starbeat! This guide will help you get started with using our documentation tool.

## Installation

```bash
bun install
```

## Quick Start

1. Create a new markdown file in `src/docs/`
2. Add frontmatter with `title` and `order`
3. Start the dev server with `bun run dev`

Your documentation will automatically appear in the sidebar!
